# 童发发的大模型学习之旅

作者：Tong Tong

B站首页：[Double童发发](https://space.bilibili.com/323109608)

## 1. 项目简介
童发发学习大模型的过程记录，会不断更新，包含代码、部分jupyter notebook讲稿等。

## 2. 项目结构
```
- README.md 说明文档
- bilibili 存在B站视频对应的jupyter讲稿
  - rope.ipynb 旋转位置编码讲稿
- MCP 测试MCP协议
  - server.py MCP服务
- manus 从零开始构建manus！
  - agent.py 智能体
  - tool_manager.py 工具管理器
  - fc_test.py 测试大模型function calling
```
